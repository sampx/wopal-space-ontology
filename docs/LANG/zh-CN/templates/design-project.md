## 6. Interfaces and Contracts

对外暴露面，规格级描述。覆盖：CLI 命令、API、事件、文件格式、schema、协议、集成契约、被消费的模板和配置。

- 每个接口列出名称、消费者、输入输出约定
- 文件格式 / schema 定义字段、约束、生成规则
- 配置契约说明层级关系和默认值
- 模板列出模板名、渲染目标、职责

若项目含前端 UI，本节作为 UI 设计契约：

- **技术栈**：框架、UI 库、构建工具
- **设计令牌**：色彩体系、间距规范、字体层级、响应式断点
- **组件规范**：组件库来源和扩展规则、受控/非受控约定
- **页面/路由结构**：页面清单、路由层级、布局模板
- **交互规范**：加载态、空态、错误态的统一处理约定
